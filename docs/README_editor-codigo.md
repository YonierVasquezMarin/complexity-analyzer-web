# Editor de Código

## ¿Qué es el editor de código?

El editor de código es la herramienta principal que permite a los usuarios escribir, modificar y visualizar su pseudocódigo de manera cómoda y eficiente.

## Funcionalidades del Editor

### Edición de Texto
El editor permite escribir y modificar pseudocódigo de forma libre, similar a un editor de texto normal pero optimizado para código.

### Visualización del Código
El código se muestra con un formato que facilita su lectura, con colores y estilos que ayudan a distinguir diferentes partes del código.

### Modo de Solo Lectura
El editor puede funcionar en modo de solo lectura, donde se muestra el código pero no se puede modificar. Esto es útil cuando se quiere ver el código sin riesgo de modificarlo accidentalmente.

## Ubicación en la Interfaz

### En la Página Principal
En la pantalla principal, el editor ocupa la parte derecha de la pantalla (70% del ancho), mientras que la lista de archivos está a la izquierda (30% del ancho).

### En Páginas de Análisis
En las páginas donde se muestran análisis, el editor puede aparecer en modo de solo lectura para mostrar el código que está siendo analizado.

## Controles del Editor

Junto al editor hay controles que permiten:
- Guardar cambios manualmente (aunque también se guardan automáticamente)
- Limpiar el contenido del editor
- Realizar acciones específicas relacionadas con el código

## Guardado Automático

El editor guarda automáticamente los cambios después de un breve período de inactividad (500 milisegundos). Esto significa que:
- No es necesario guardar manualmente
- Los cambios se preservan incluso si se cierra la aplicación
- Se evita la pérdida de trabajo por olvido

## Experiencia de Usuario

El editor está diseñado para ser:
- **Intuitivo**: Fácil de usar sin necesidad de instrucciones
- **Rápido**: Responde inmediatamente a las acciones del usuario
- **Confiable**: Guarda automáticamente para evitar pérdida de datos
